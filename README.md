# Serverless-ESA

## 简介

基于阿里云 ESA「函数与Pages」 能力，构建的 API 模板项目。

目前已经实现了随机图片API功能，具体参考[random-img/README.md](./random-img/README.md)

## 特性

- 边缘函数运行，低延迟、免运维
- 使用 EdgeKV 管理配置
- 灵活的路由注册机制，易于扩展新功能
